Recursos Utilizados para el Proyecto

Reactjs

Esta herramienta sera usada para el frontend, es decir para todo aquello donde los usuarios podran interactuar con la pagina web.

Django

Este framework sera usado para el backend, es decir todos los procesos que van detras del como el usuario interactua con la pagina web, por lo que se habla del manejo de la base de datos de la pagina web.

Firebase

Base de datos no relacional que sera usada para esta pagina web.

BootstrapMade

Esta herramienta sirve como una plantilla de como se vera la pagina we, a su ves ahorra el tener que hacer todo el codigo HTML, CSS y JS desde cero para la pagina web.
